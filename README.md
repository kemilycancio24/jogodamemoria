Jogo da Memória — Harry Potter

Este é um jogo da memória simples e divertido inspirado no universo de Harry Potter. O objetivo é encontrar todos os pares de cartas iguais, testando sua atenção e memória enquanto se diverte com personagens e elementos mágicos do mundo bruxo. Basta clicar nas cartas para virá-las e tentar descobrir onde estão os pares. Quando todos forem encontrados, o jogo é concluído. O projeto foi desenvolvido com HTML, CSS e JavaScript, sendo leve, intuitivo e ideal para quem está começando a programar ou gosta de desafios rápidos no navegador. Sinta-se à vontade para explorar, modificar e aprimorar!
